### Features
-  Fetches fruit data from a custom-built Express/Node.js backend
-  Displays the list dynamically using React and React Hooks (`useEffect`, `useState`)]
### Frontend:
- **React (Vite)**
- **JavaScript**
- **CSS**

### Backend:
- **Node.js**
- **Express.js**
- **CORS** for cross-origin requests
- **REST API** with JSON response
